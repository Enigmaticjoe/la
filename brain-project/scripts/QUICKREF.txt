  "SELECT task_category, ROUND(temperature,1), AVG(quality_rating), COUNT(*)

# Quality trend over time

🐛 TROUBLESHOOTING
───────────────────────────────────────────────────────────────────────────────

No optimizations happening?
  → Check you have 10+ rated conversations
  → Verify conversations have quality ratings in OpenWebUI
  → Review logs: tail -f data/evolution/evolution.log

GPU monitoring not working?
  → Ensure nvidia-smi is accessible
  → Add --gpus all to docker run
  → Check NVIDIA driver installation

Backup failing?
  → Verify disk space: df -h
  → Check permissions on backup directory
  → Review: cat backups/backup.log

Evolution container not starting?
  → Check logs: docker compose logs evolution
  → Verify network: docker network ls | grep brain
  → Ensure dependencies running: docker compose ps
