🐛 TROUBLESHOOTING
───────────────────────────────────────────────────────────────────────────────

Problem: No optimizations happening after 1 week
Solution:
  1. Check you have rated conversations in OpenWebUI (need quality ratings!)
  2. Verify you have 10+ conversations
  3. Review logs: tail -f ../data/evolution/evolution.log
  4. Query database: sqlite3 ../data/evolution/evolution.db "SELECT COUNT(*) FROM metrics;"

Problem: GPU monitoring not working
Solution:
  1. Check nvidia-smi is accessible: nvidia-smi
  2. Verify GPU access in docker-compose (uncomment deploy section)
  3. Check NVIDIA driver: nvidia-smi --version

Problem: Evolution container not starting
Solution:
  1. Check logs: docker compose logs evolution
  2. Verify network: docker network ls | grep brain
  3. Ensure dependencies running: docker compose ps
  4. Check .env file has required variables

Problem: Backup failing
Solution:
  1. Check disk space: df -h
  2. Verify permissions: ls -la ../backups
  3. Review backup log: cat ../backups/backup.log
  4. Ensure services are running
